## Overview

The Footer component provides the bottom section of the portfolio with social media links, copyright information, and includes a TypeScript script for time display functionality.

```astro
            <a class="nav-link fw-medium p-0 text-secondary" href="/projects">
                <i class="bi bi-arrow-right"></i>
                Linkedin
            </a>
        </li>
        <li class="nav-item">
            <a class="nav-link fw-medium p-0 text-secondary" href="/about">
                <i class="bi bi-arrow-right"></i>
                Github
            </a>
        </li>
        <li class="nav-item">
            <a class="nav-link fw-medium p-0  text-secondary" href="/blog">
                <i class="bi bi-arrow-right"></i>
                Email
            </a>
        </li>
    </ul>
    <p class="m-0 mb-2">&copy; Diseñado y desarrollado por Ronald Santana 2025</p>

<script>
    function actualizarHora(): void {
        const ahora = new Date();
        const hora: string = ahora.toLocaleTimeString();

        if (horaElem) {

```

## TypeScript Script

The component includes a TypeScript function for updating time:

```typescript
function actualizarHora(): void {
    const ahora = new Date();
    const hora: string = ahora.toLocaleTimeString();
    const horaElem = document.getElementById("hora");
    if (horaElem) {
        horaElem.textContent = hora;
    }
}

actualizarHora();
setInterval(actualizarHora, 1000);
```

### How It Works

1. **Function Definition**: `actualizarHora()` (Spanish for "update time")
2. **Get Current Time**: Uses `new Date()` and `toLocaleTimeString()`
3. **Find Element**: Searches for an element with `id="hora"`
4. **Update Display**: Sets the text content to the current time
5. **Initial Call**: Runs once immediately
6. **Interval**: Updates every 1000ms (1 second)

<Note>
  The script references an element with `id="hora"` that doesn't exist in the current component markup. This suggests the time display feature may have been removed or is intended for a different section.
</Note>

### Adding Time Display

To use the time functionality, add this to the footer:

```astro
<footer class="my-5 py-5 animate-on-load animate-delay-4">
    <ul class="nav gap-4 mb-2">
        <!-- Links here -->
    </ul>
    <p class="m-0 mb-2">&copy; Diseñado y desarrollado por Ronald Santana 2025</p>
    <p class="text-secondary small">
        Current time: <span id="hora"></span>
    </p>
</footer>
```

## Accessibility

<Tip>
  The footer uses semantic HTML (`<footer>`, `<nav>`, `<ul>`) which helps screen readers understand the page structure.
</Tip>

### Improvements for Accessibility

1. Add `aria-label` to the navigation:

```astro
<ul class="nav gap-4 mb-2" aria-label="Social media links">
```

2. Add screen reader text for icon-only indicators:

```astro
<a class="nav-link fw-medium p-0 text-secondary" href="...">
    <i class="bi bi-arrow-right" aria-hidden="true"></i>
    <span>Linkedin</span>
</a>
```

## Responsive Design

The footer is responsive by default:
- Uses Bootstrap's responsive spacing utilities
- `gap-4` maintains consistent spacing on all screen sizes
- Navigation items wrap naturally on smaller screens

<Tip>
  For better mobile experience, consider using flexbox utilities:
</Tip>

```astro
<ul class="nav gap-4 mb-2 flex-column flex-md-row">
```

This stacks links vertically on mobile and horizontally on medium+ screens.
