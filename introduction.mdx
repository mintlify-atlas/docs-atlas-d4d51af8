## Overview

The **RSantana Portfolio** is a modern, high-performance personal portfolio website showcasing web development projects and skills. Built with Astro's static site generation and styled with Bootstrap 5, this portfolio demonstrates best practices in modern web development while maintaining simplicity and performance.

The portfolio features a clean, minimalist design with smooth animations and a responsive layout that works seamlessly across all devices.

## Key Features

### Content Collections
Leverage Astro's powerful content collections system for managing projects. Each project is defined with type-safe frontmatter including:
- Title and description
- Publication and update dates
- Hero images
- Project tags
- External project URLs

### Bootstrap 5 Integration
Fully integrated Bootstrap 5 framework providing:
- Responsive grid system
- Pre-built components (breadcrumbs, navigation)
- Bootstrap Icons for visual elements
- Mobile-first responsive design

### Intersection Observer Animations
Smooth scroll-triggered animations using the native Intersection Observer API:
- Fade-in-up effects for project cards
- Staggered animation delays for visual interest
- Performance-optimized with automatic observer cleanup

### Astro Static Site Generation
Benefit from Astro's modern architecture:
- Zero JavaScript by default for optimal performance
- Static site generation for fast page loads
- Dynamic routing for project pages
- View transitions for smooth navigation

### Project Showcase with Dynamic Routing
Automatic project page generation using Astro's file-based routing:
- Dynamic `[...slug]` routes for individual projects
- Breadcrumb navigation for better UX
- Content rendering from Markdown files

## Architecture

The portfolio follows Astro's recommended project structure:

```plaintext
src/
├── components/      # Reusable Astro components
│   ├── Card.astro
│   ├── Header.astro
│   ├── Nav.astro
│   ├── Projects.astro
│   └── Footer.astro
├── content/         # Content collections
│   ├── config.ts    # Collection schemas
│   └── projects/    # Project markdown files
├── layouts/         # Page layouts
│   └── Layout.astro
└── pages/           # File-based routing
    ├── index.astro
    ├── projects.astro
    └── projects/[...slug].astro
```

## Design Philosophy

The RSantana Portfolio embraces several key principles:

- **Performance First**: Static generation and minimal JavaScript ensure fast load times
- **Content-Focused**: Clean, minimalist design puts your projects front and center
- **Accessibility**: Semantic HTML and proper ARIA labels for screen readers
- **Maintainability**: Type-safe schemas and component-based architecture
- **Internationalization**: Currently in Spanish, easily adaptable to other languages
